You need [Node.js](https://nodejs.org/en/download/) and [Tailwind CSS](https://tailwindcss.com/docs/installation/) installed.


1. install:
```bash
npm i --save tailwindcss-classless
```

2. Then add tailwindcss-classless to your `tailwind.config.js` files:

```javascript
/** @type {import('tailwindcss').Config} */
export default {
  content: ['./src/**/*.{astro,html,js,jsx,md,mdx,svelte,ts,tsx,vue}'],
  plugins: [
    require('tailwindcss-classless'),
  ],

  theme: {
    extend: {
      // This follows the 60-30-10 rule of design
      // https://medium.com/rutgers-creative-x/understanding-color-for-ui-design-ec53719e880e

      // backgrounds and main elements
      surface: ({ theme }) => theme('colors.neutral'),

      // 2nd level supporting elements text, icons
      foreground: ({ theme }) => theme('colors.neutral'),

      // colors for active elements or call to actions
      accent: ({ theme }) => theme('colors.blue'),
    }
  }
}
```

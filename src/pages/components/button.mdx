A dead simple CSS rule for styling buttons by default

## CSS

<Code lang='postcss' code={ButtonCSS} />

## Usage

<CodePreview 
  class="not-prose"
  code={`
<button>Click me!</button>
<input type='submit' />
  `}
  lang='html'
>
  <div class='space-x-4'>
    <button>Click me!</button>
    <input type='submit' />
  </div>
</CodePreview>

## Variants

You can continue to use all DaisyUI variants to style your buttons as needed with classes.

<CodePreview 
  class="not-prose"
  code={`
<button class='btn-primary'>Click me!</button>
<button class='btn-outline btn-success'>Click me!</button>
  `}
  lang='html'
>
  <div class='space-x-4'>
    <button className='btn-primary'>Click me!</button>
    <button className='btn-outline btn-success'>Click me!</button>
  </div>
</CodePreview>

## Form buttons

inputs with type attributes like `type="submit"` and `type="button"` are also displayed as buttons.

<CodePreview 
  class="not-prose"
  code={`
<input type="submit" />
<input type="reset" />
<input type="button" value="Button" />
  `}
  lang='html'
>
  <div class='space-x-4'>
    <input type="submit" />
    <input type="reset" />
    <input type="button" value="Button" />
  </div>
</CodePreview>

## Disabled

Buttons come with a disabled style.



<CodePreview 
  class="not-prose"
  code={`
<button disabled>Disabled</button>
<button disabled class='btn-primary'>Disabled</button>
<button disabled class='btn-secondary'>Disabled</button>
  `}
  lang='html'
>
  <div class='space-x-4'>
    <button disabled>Disabled</button>
    <button disabled className='btn-primary'>Disabled</button>
    <button disabled className='btn-secondary'>Disabled</button>
  </div>
</CodePreview>
